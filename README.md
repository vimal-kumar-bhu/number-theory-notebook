Number_Theory_Concepts_Proofs
## 📘 Topics Covered

This repository includes core topics of Number Theory:

- Greatest Common Divisor (GCD) and Euclidean Algorithm  
- Modular Arithmetic and Fast Modular Exponentiation  
- Euler’s Totient Function and its properties  
- Chinese Remainder Theorem (system of congruences)  
- Prime Numbers: testing and generation techniques  
Pure Number Theory project combining theory, proofs, and computational implementations in Python.
